## Getting Started

First, install packages:

```bash
npm -i
# or
yarn install
```

Create a .env.local file

```bash
NEXT_PUBLIC_RPC=https://api.devnet.solana.com
NEXT_PUBLIC_NETWORK=devnet
```

And then run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
